# 🏦 Kredit və Bank Məhsullarının Analitikası Dashboard

Bu layihə bank sektorunda kredit portfelinin idarə edilməsi, faiz dərəcələrinin optimallaşdırılması və filial performansının analizi üçün hazırlanıb.

## 🚀 Əsas Biznes Göstəriciləri (KPIs)
- **Kredit Həcmi:** 187M AZN istehlak və 237M AZN lombard krediti.
- **Filial Analizi:** Yasamal (25.3M AZN) və MXD (23.2M AZN) filiallarının liderliyi.
- **Faiz Səviyyələri:** Portfelin böyük hissəsinin (126K əməliyyat) yüksək faiz dərəcələri ilə idarə olunması.
- 
- ## 🔍 Analiz Metodologiyası
Bu layihədə mən aşağıdakı mərhələləri tətbiq etmişəm:
- **Data Cleaning:** Power Query vasitəsilə "null" dəyərlərin və xətalı dataların təmizlənməsi.
- **AI Insights:** "Key Influencers" vizualı ilə faiz dərəcəsinin azalmasına təsir edən faktorların (məs. Kredit kartı məhsulu) analizi.
- **Geospatial Analysis:** Filialların coğrafi mövqeyinə görə kredit qalıqlarının paylanması.

## 🛠 Texniki Analiz (Süni İntellekt və Data Science Yanaşması)
- **Key Influencers:** Faiz dərəcəsinin azalmasına təsir edən əsas faktorların analizi (məsələn, Kredit kartı məhsulunun təsiri).
- **Decomposition Tree:** Faiz və komissiya gəlirlərinin filiallar və məhsullar üzrə iyerarxik dekompozisiyası.
- **Geospatial & Trend Analysis:** Filiallar üzrə qalıq borc trendinin və borcalanların cins bölgüsünün tədqiqi.
